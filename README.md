# 🕰️ 時影雙行 Echo Trail

一個基於時間操控和自我對話的哲學解謎遊戲。在斷裂的時序都市「納赫拉姆」中，與你的過去協作，尋找被遺忘的真相。

## 🎮 遊戲特色

- **殘影系統**: 創建過去的自己，與Echo協作解謎
- **深度劇情**: 探索自我、時間與命運的哲學主題
- **多重結局**: 根據選擇獲得不同的故事結局
- **創新玩法**: 時間回放機制與解謎元素結合
- **情感敘事**: 與過去的自己對話，面對內心的掙扎

## 🚀 快速開始

1. 下載或克隆此專案
2. 使用現代瀏覽器打開 `index.html`
3. 點擊「開始遊戲」開始你的時影之旅

## 🎯 操作說明

### 鍵盤控制
- **WASD** 或 **方向鍵**: 移動角色
- **空白鍵** 或 **Q**: 創建殘影
- **E**: 互動
- **Enter**: 確認對話
- **Esc**: 選單

### 滑鼠控制
- **左鍵點擊**: 選擇/互動
- **右鍵點擊**: 創建殘影
- **滾輪**: 縮放視角

### 觸控控制
- **拖拽**: 移動角色
- **點擊**: 互動
- **長按**: 創建殘影

## 📖 遊戲章節

### 第一章：殘影初醒
學會創建並配合Echo，發現這些Echo不只是錄像，而是曾經的你「正在活著」。

### 第二章：記憶交錯
出現需要多段Echo合作的解謎，在不同殘影的視角中看見自己曾經的錯誤與懦弱。

### 第三章：重演無法改變的命運
Echo協助你通過重重障礙，但某些痛苦的記憶始終無法被改變。

### 第四章：時序審判
Echo們分裂為兩派，你被迫在「自我忠誠」與「自我犧牲」間做選擇。

### 第五章：你，與我
最後一關，Echo不再只是協作，而是變成Boss與你對決。

## 🎨 技術架構

### 模組化設計
```
js/
├── main.js              # 主入口點
├── core/
│   └── GameEngine.js    # 遊戲核心引擎
├── entities/
│   └── Player.js        # 玩家角色
├── systems/
│   └── EchoSystem.js    # 殘影系統
├── ui/
│   └── UIManager.js     # UI管理器
├── audio/
│   └── AudioManager.js  # 音頻管理器
├── save/
│   └── SaveManager.js   # 存檔管理器
├── levels/
│   └── LevelManager.js  # 關卡管理器
├── input/
│   └── InputHandler.js  # 輸入處理器
└── physics/
    └── PhysicsEngine.js # 物理引擎
```

### 核心系統

- **遊戲引擎**: 統一管理所有系統，處理遊戲循環
- **殘影系統**: 記錄玩家行為，創建可互動的過去版本
- **物理引擎**: 碰撞檢測、運動模擬
- **關卡管理**: 動態生成關卡，管理遊戲進度
- **音頻系統**: 3D音效、背景音樂、合成音效
- **存檔系統**: 自動保存、多插槽存檔、匯入匯出

## 🎵 音效設計

遊戲包含完整的音效系統：
- **UI音效**: 選單操作、對話顯示
- **遊戲音效**: 殘影創建、互動、受傷
- **環境音效**: 時間扭曲、回聲低語
- **背景音樂**: 各章節主題音樂

## 💾 存檔功能

- **自動保存**: 每30秒自動保存進度
- **多插槽**: 支援5個存檔插槽
- **存檔資訊**: 顯示章節、時間、截圖
- **匯入匯出**: 支援存檔文件的匯入匯出

## 🎯 遊戲機制

### 殘影系統
1. **創建殘影**: 消耗能量創建當前位置的殘影
2. **行為記錄**: 殘影會重播你之前的行動
3. **協作解謎**: 利用殘影觸發開關、移動平台
4. **對話互動**: 與殘影對話，探索內心世界

### 解謎元素
- **時序謎題**: 需要特定順序觸發的機關
- **協作謎題**: 需要多個殘影同時行動
- **記憶碎片**: 收集散落的記憶，了解故事背景
- **道德選擇**: 影響劇情走向的關鍵決定

## 🌟 特殊功能
